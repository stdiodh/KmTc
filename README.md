## 경민IT고등학교 기능반 홍보 홈페이지
고등학교 1학년 시절 기능반에 합격해서 개발을 처음 접했을 때 만든 홈페이지입니다. </br>
해당 홈페이지는 기능반을 추후 학년들에게 홍보하고 싶어 기획하고 만들게 되었습니다.
</br>
</br>
